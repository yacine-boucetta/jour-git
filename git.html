Tableau de fonction git
Nom Fonction
Git clone Créer un pont de communication entre un fichier local et un git hub via une URL
Git commit Permet de précharger un fichier pour l'envoyer sur le github
Git push Permet d'envoyer la mise à jour du fichier dans la branche du github
Git pull Permet d'extraire un fichier du github et de fusionner cette version à celle du fichier local correspondant
Git merge Permet de fusionner les fichier correspondants
Git checkout La commande git checkout est utilisée pour passer d’une branche à l’autre et en extraire le contenu dans votre répertoire de travail.
Git show Permet de visualiser un fhichier et l'historique des differents commit
